# EEGP
Easy-to-use EEG preprocessing pipeline based on MNE

## Description
本项目主要是针对不同的 EEG 数据集，基于 mne 实现了一个简单可复用的数据预处理 pipeline。
目前这个项目还比较简陋，适合科研新人了解 EEG 数据预处理的流程以及如何借助 mne 对数据进行预处理。
